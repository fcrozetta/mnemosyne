# Claim 0002: Local and Deploy Infra First

## Statement

After directives are locked, local and deploy infrastructure contracts are defined before broader feature work.

## Scope

- Local runtime assumptions (developer workflows, compose/devcontainer intent).
- Deploy assumptions (target runtime, boundaries, non-goals).
- Minimal, explicit infrastructure posture.

## Acceptance Checks

- Infra contract docs exist and are reviewed.
- Phase gate established before API/MCP implementation begins.
