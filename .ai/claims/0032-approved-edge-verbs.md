# Claim 0032: Approved Edge Verbs

## Statement

The graph vocabulary accepts `owns`, `located_at`, `applies_to`, and `targets`
as valid concise verb-style edge names.

## Scope

- These verbs are allowed for current schema design.
- Future naming should stay concise and verb-oriented.

## Acceptance Checks

- Schema docs use these verbs consistently where the semantics fit.
- Future diagrams do not expand them into longer redundant phrases.
