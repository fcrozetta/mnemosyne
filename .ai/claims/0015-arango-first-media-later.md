# Claim 0015: Arango Ingest First, Media Later

## Statement

Initial implementation focuses on ingesting notes/events into ArangoDB using
basic collections before media storage integration.

## Scope

- Deliver core ingest path into database.
- Defer media artifact storage backend integration.

## Acceptance Checks

- Basic ingest collections are defined for v0.
- Media storage implementation is explicitly tracked as future phase.
