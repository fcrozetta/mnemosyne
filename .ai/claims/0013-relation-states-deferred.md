# Claim 0013: Deferred Relation Verification States

## Statement

Relation states `unverified` and `rejected` are deferred from v0.

## Scope

- v0 remains verified-only.
- Later phases add reviewable verification transitions.

## Acceptance Checks

- Roadmap includes support for `unverified` and `rejected`.
- Migration path from verified-only is documented before rollout.
